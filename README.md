# Virtual Try-On with Nano Banana Pro 
This repository is an implementation of Virtual Try-On using Gemini Nano Banana Pro that allows users to upload person's and apparel's image.

## Run the Web App:

1. Clone the repository on your local machine.
2. Navigate to `cd Virtual-Try-On-Gemini` directory.
3. Run `pip install -r requirements.txt` to install the packages.
4. Open `.env` file and configure your `GOOGLE_API_KEY`.
5. Run `flask run` to start the server.
6. Open `localhost:5000` on your web browser to bring up the web UI.
